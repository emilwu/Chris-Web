## 格式說明

每個任務條目應包含：
- **任務接收時間：** [日期和時間]
- **任務說明：** 要執行的任務摘要
- **子任務：** 將複雜任務分解為較小的子任務
  - **行動：** 要執行的特定指示或步驟
  - **依賴項或先決條件：** 開始任務前必須滿足的任何條件
  - **狀態：** 進行中 / 已完成 / 已封鎖
  - **註：** 關於進度、任何障礙或其他見解的註釋
- **完成日期：** 任務完成的日期和時間
